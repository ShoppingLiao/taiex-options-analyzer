## 🎯 統一後的設計原則

### 1. 字體大小層級

```
2rem      - 超大數值（結算價）
1.3rem    - 區塊標題 (h3)
1.2rem    - 關鍵數據值
0.95rem   - 次級標題
0.9rem    - 次要資訊
0.85rem   - 主要內容文字
0.75rem   - 極小輔助文字
```

### 2. 顏色規範

```css
/* 主要文字 */
color: #3c3c3c; /* 一般內容 */
color: #3b0764; /* 紫色系內容 */
color: #78350f; /* 黃色系內容 */
color: #92400e; /* 橙色系標題 */

/* 數據顏色 */
color: #2563eb; /* 藍色 - 一般數據 */
color: #059669; /* 綠色 - 正面數據 */
color: #d97706; /* 橙色 - 警示數據 */
color: #dc2626; /* 紅色 - 風險提示 */
```

### 3. 間距規範

```css
/* Padding */
padding: 15px; /* 卡片內容 */
padding: 20px; /* 中型區塊 */
padding: 25px; /* 大型區塊 */

/* Margin */
margin-bottom: 8px; /* 小間距 */
margin-bottom: 10px; /* 中間距 */
margin-bottom: 15px; /* 大間距 */
```
