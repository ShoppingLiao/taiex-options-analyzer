# 🚀 結算日報告快速上手指南

## 📋 概述

結算日報告系統每週產生 **2 份報告**，幫助你預測週三和週五的結算區間。

## 🎯 核心概念

### 為什麼是週三和週五？

台指選擇權通常在**每月第三個週三**結算，但每週的週三和週五是重要的觀察點：

- **週三**：半週結算觀察點
- **週五**：週末前最後交易日

### 分析週期

```
週一 ─┐
週二 ─┤→ 產生「週三結算預測」
      │
週三 ─┤
週四 ─┤→ 產生「週五結算預測」
週五
```

## 🛠️ 實戰操作

### Step 1: 準備單日報告數據

首先要有基礎的單日報告：

```bash

python3 main.py --date 20260105

# 產生週二報告
python3 main.py --date 20260106
```

### Step 2: 生成週三結算預測

## 📊 如何解讀報告

### 1. 趨勢分析總覽

```
┌─────────────────────────────────┐
│ 整體趨勢: 多頭趨勢 (強勁)       │
│ 強度評級: ⭐⭐⭐⭐⭐            │
│ 預測區間: 23,400 ~ 24,200      │
└─────────────────────────────────┘
```

**解讀：**

- **趨勢方向**：多頭/空頭/震盪
- **強度評級**：1-5 星，越多星越強
- **預測區間**：結算可能的價格範圍

### 2. 趨勢訊號

```
📈 [4/5] 買權 OI 持續增加 (+5,000)，多方積極佈局
```

**解讀：**

- **📈 圖示**：多頭訊號（📉 空頭、➖ 中性）
- **[4/5]**：訊號強度，5 最強
- **說明**：訊號的具體含義

**訊號類型：**

1. **OI 變化**：觀察買權/賣權的未平倉變化
2. **P/C Ratio**：市場情緒指標
3. **價格動能**：趨勢方向與力道
4. **Max Pain**：磁吸效應

### 3. 結算劇本

```
🚀 強勢上攻 (55.0%)
區間: 24,150 ~ 24,450
條件: ✓ 多方持續加碼...
策略: 順勢做多，停利 +100，停損在 Max Pain
```

**解讀：**

- **機率 55%**：發生可能性（最高 60%）
- **區間**：該劇本的結算範圍
- **條件**：劇本成立的前提
- **策略**：操作建議（僅供參考）

**劇本類型：**

- 🚀 **強勢上攻**：多頭力道強，向上突破
- ⚖️ **震盪整理**：多空平衡，區間操作
- 📉 **回檔修正**：空頭力道強，向下修正
- 🧲 **Max Pain 磁吸**：價格向最大痛點靠攏

### 4. 關鍵指標

```
Max Pain: 23,500
P/C Ratio: 0.85 (-0.05 ↓)
買權 OI: 50,000 (+3,000 ↑)
賣權 OI: 42,500 (-1,500 ↓)
```

**解讀：**

- **Max Pain**：莊家最有利的價位
- **P/C Ratio**：< 0.8 偏多，> 1.2 偏空
- **OI 變化**：正值增加，負值減少

### 5. 風險提示

```
⚠️ P/C Ratio 極低 (0.65)，市場過度樂觀
⚠️ 價格遠離 Max Pain (500 點)，磁吸風險高
```

**解讀：**

- 關注市場異常狀況
- 提醒可能的風險因素
- **重要**：僅供參考，不構成投資建議

## 💡 使用技巧

### 技巧 1: 觀察訊號一致性

當多個訊號方向一致時，趨勢較為明確：

```
📈 [4/5] OI 變化 → 多頭
📈 [4/5] P/C Ratio → 多頭
📈 [5/5] 價格動能 → 多頭
-----------------------------
整體趨勢: 多頭 (強勁) ⭐⭐⭐⭐⭐
```

### 技巧 2: 劇本機率差異

機率相近時，市場不確定性高：

```
🚀 強勢上攻 (35%)
⚖️ 震盪整理 (33%)
📉 回檔修正 (32%)
→ 建議: 觀望為主，避免重押
```

機率差距大時，趨勢較明確：

```
🚀 強勢上攻 (60%)
⚖️ 震盪整理 (25%)
📉 回檔修正 (15%)
→ 建議: 順勢操作，設好停損
```

### 技巧 3: Max Pain 磁吸效應

當價格距離 Max Pain 很遠時：

```
當前價格: 24,500
Max Pain: 23,500
距離: +1,000 點
→ 結算前可能回歸 Max Pain
```

### 技巧 4: 結合兩份報告

週三和週五報告互相參考：

```
週三預測: 多頭趨勢 → 結算 23,800
實際結算: 23,750
週五預測: 依週三結果調整
```

## ⚠️ 重要提醒

### 1. 數據限制

- 預測基於**歷史數據**
- 無法預測**突發事件**
- 僅供**參考**，非投資建議

### 2. 風險控管

- **設定停損**：避免過度虧損
- **分批進場**：降低風險
- **控制倉位**：不要過度槓桿

### 3. 市場變化

- 結算前市場可能劇烈波動
- 大戶可能刻意調節部位
- 外部因素（新聞、政策）影響

### 4. 正確心態

```
✅ 參考報告，輔助判斷
✅ 設定停損，控制風險
✅ 保持理性，避免貪婪

❌ 盲目跟單
❌ 重押單邊
❌ 忽略風險
```
