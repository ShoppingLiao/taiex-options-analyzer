## 🚀 未來使用指南

### 每日報告生成（自動使用正確樣式）

```bash
# 1. 添加台指資料（如需新日期）
# 編輯 src/parser.py 的 tx_data_map

# 2. 放置 PDF
# 放入 data/pdf/

# 3. 生成報告
python3 main.py --date 20260113

# 4. 更新首頁
python3 generate_index_with_weekday.py
```

### 結算日報告生成（自動使用正確樣式）

```bash
# 週三結算
python3 generate_settlement_report.py wed

# 週五結算
python3 generate_settlement_report.py fri

# 更新首頁
python3 generate_index_with_weekday.py
```

### 樣式更新（正確做法）

```bash
# 1. 修改模板
# 編輯 templates/report.html 或 templates/settlement_report.html

# 2. 重新生成報告
python3 main.py  # 或其他生成腳本

# 3. 新報告自動使用新樣式
# 無需手動修復或執行批量更新腳本
```

### ⚠️ 不應該做的事

```bash
# ❌ 不要執行這些腳本
python3 maintenance/scripts/update_section_content.py
python3 maintenance/scripts/add_section_content_css.py
python3 maintenance/scripts/update_ai_performance_layout.py
python3 maintenance/scripts/update_border_radius.py

# ❌ 不要手動修改已生成的 HTML
# ❌ 不要複製舊報告並修改內容
```
